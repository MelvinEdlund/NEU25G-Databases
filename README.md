# NEU25G-Databases

Samling av material för kursen **Databases (NEU25G)**.

## [Code-Along](https://github.com/MelvinEdlund/NEU25G-Databases/tree/main/Code-Along)
Kortare genomgångar och code-alongs från lektioner. Varje mapp motsvarar ett moment/lektion och innehåller exempelprojekt.

## [Exercises](https://github.com/MelvinEdlund/NEU25G-Databases/tree/main/Exercises)
Övningsuppgifter för att repetera och fördjupa innehållet från föreläsningar och code-alongs.

## [MongoDB](https://github.com/MelvinEdlund/NEU25G-Databases/tree/main/MongoDB)
Exempel och anteckningar kopplade till MongoDB (queries, aggregation, modellering m.m.).

## [SQL](https://github.com/MelvinEdlund/NEU25G-Databases/tree/main/SQL)
SQL-relaterat material: DDL/DML-exempel, joins, constraints, normalisering och andra databaskoncept.
